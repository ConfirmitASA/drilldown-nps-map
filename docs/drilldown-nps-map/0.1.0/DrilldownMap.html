DrilldownMap

new DrilldownMap(source, tableID, rowheaders, hierarchy, initMapopt, containerID, mappointCallback, colorFn, valueColumnopt, fullParentLevelInfoopt, pageStateIdopt, normals, normalsSeparatoropt, optionsopt)

Source:
  • DrilldownMap.js, line 39

Creates a drilldown map. It uses a color function colorFn that allows colorcode countries colorFn accepts two attributes: value and target and must return a color string based on those two attributes. Make sure hierarchy has target loaded from DBDesigner table into each hierarchy level, otherwise a default config dataClasses takes precedence on value Example:

{ //some constructor configuration above colorFn: function(value,target){ return (value!=null && target!=null)? (value - target >= 0) ? "#18BC9C" : ((value >= 0.9*target) ? "#FF4900" : "#E45335") : undefined; } //some constructor configuration below }
